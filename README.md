# 📌 Overview

Smart Event Platform is a Full Stack web application developed to manage institutional events through a secure and scalable architecture.

The platform allows users to create, manage and visualize events while providing administrators with validation workflows, approval systems and centralized event control.

The application was developed using:

- **Spring Boot** for backend architecture
- **Vue.js** for frontend interface
- **JWT Authentication** for secure access control
- **SQL Database** for persistent data management
- **REST APIs** for frontend-backend communication

# ✨ Main Features

## 🔐 Authentication & Security
- JWT-based authentication system
- Secure login and session management
- Role-based access control
- Protected API routes

---

## 👥 User Roles
### Administrator
- Approve or reject event proposals
- Manage platform events
- Control event visibility and states

### Teachers / Users
- Create event proposals
- Submit events for approval
- Manage assigned events

---

## 📅 Event Management
- Institutional event calendar
- Event filtering and organization
- Real-time event visibility
- Event lifecycle management

### Event States
- Pending
- Confirmed
- Rejected

---

## 📂 CSV Import System
The platform includes a CSV import module that allows bulk event uploads directly into the calendar system.

This feature streamlines institutional event management and automates event registration processes.

---

## 📺 TV Display Mode
Special display mode designed for institutional screens and TV panels.

Features:
- Automatic event visualization
- Clean responsive layout
- Public event broadcasting

# 🚀 Getting Started

## Clone repository

```bash
git clone https://github.com/filotea25/smart-event-platform.git
```

---

## Backend Setup

```bash
cd backend/agenda
mvn spring-boot:run
```

Backend runs on:

```text
http://localhost:8080
```

---

## Frontend Setup

```bash
cd frontend/frontend
npm install
npm run dev
```

Frontend runs on:

```text
http://localhost:5173
```
